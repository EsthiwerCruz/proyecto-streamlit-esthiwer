# Proyecto Módulo 1 – Fundamentos de Programación

## Descripción

Aplicación interactiva desarrollada en **Streamlit** que integra conceptos fundamentales de programación aprendidos durante el Módulo 1 del curso Python for Analytics.

El proyecto incluye:

* Variables
* Condicionales
* Estructuras de datos
* Funciones
* Programación funcional (map y lambda)
* Programación Orientada a Objetos (POO)

La aplicación permite gestionar actividades financieras, evaluar presupuestos y calcular retornos esperados.

---

## Estructura del proyecto

* `app.py`
* `requirements.txt`
* `README.md`

---

## Instrucciones de ejecución

1. Instalar dependencias:

```
pip install -r requirements.txt
```

2. Ejecutar la aplicación:

```
streamlit run app.py
```
